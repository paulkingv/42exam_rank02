Nom de l'exercice  : rev_print
Fichiers attendus  : rev_print.c
Fonctions autorisées : write
--------------------------------------------------------------------------------

Écrivez un programme qui prend une chaîne de caractères et affiche cette chaîne à l'envers,
suivie d'une nouvelle ligne.

Si le nombre de paramètres n'est pas égal à 1, le programme affiche une nouvelle ligne.

Exemples :

$> ./rev_print "zaz" | cat -e
zaz$
$> ./rev_print "dub0 a POIL" | cat -e
LIOP a 0bud$
$> ./rev_print | cat -e
$
